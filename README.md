# Test Task: Select Player Demo

Unity / C# test task project created as part of a job application process.

---

## About

**Select Player Demo** is a small Unity demo focused on player selection flow and basic project architecture for a test assignment.

The project includes a simple selection-oriented experience with boot/loading logic, service-based organization, save-related functionality, ScriptableObject-based data setup and core systems required for the assignment.

It was developed under limited time constraints with the goal of completing the required assignment scope in a clear, practical and functional way.

---

## Project Scope

This repository represents an archived test task completed during a hiring process.

It demonstrates fast Unity/C# development under time pressure, including:

- player selection flow;
- boot/loading logic;
- service-based project organization;
- save-related logic;
- ScriptableObject-based configuration;
- prefab-based setup;
- scene flow organization;
- basic architecture suitable for a small demo.

The project should be viewed as a focused test-task implementation, not as a full-scale production project or a complete reflection of my current development standards.
